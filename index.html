HPO Portal


Search tool for the Human Phenotype Ontology (HPO) database.

How to Use This Tool ?

  • Start typing in the search box to find HPO terms.
  • Clickon the icon to get more information.
  • Clickon a term to add it to your selection.
  • Remove terms using the trash icon.
  • Export your selection as a text file.
  • All data is fetched in real-time from the HPO database.
